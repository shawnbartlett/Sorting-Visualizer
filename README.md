# Sorting Visualizer UNITY

A simple Unity project to practice and visually demonstrate sorting algorithms.

![Smooth Lerp Animaton](Devlogs/Sort_1.gif)

- Generates random integer array
- Spawns bars representing values
- Smoothly animates toward sorted postion
- Currently supports basic system sort 

Controls:
Press S to sort

## Sorting Methods (In Progress)

- EX. Bubble Sort
- EX. Insertion Sort
- EX. Sleep Sort
